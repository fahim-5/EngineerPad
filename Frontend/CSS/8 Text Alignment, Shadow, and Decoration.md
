# CSS Text Alignment, Shadows & Decorations

## Text Alignment (`text-align`)

Controls horizontal text alignment:

```css
.left-align {
  text-align: left; /* Default for most languages */
}

.center-align {
  text-align: center; /* Common for headings */
}

.right-align {
  text-align: right; /* Useful for numbers */
}

.justified {
  text-align: justify; /* Stretches to fill width */
  hyphens: auto; /* Add hyphenation for better results */
}
```

## Vertical Alignment (`vertical-align`)

Aligns inline/inline-block elements:

```css
.baseline {
  vertical-align: baseline; /* Default */
}

.middle {
  vertical-align: middle; /* Centers with surrounding text */
}

.top {
  vertical-align: top; /* Aligns with line box top */
}

img.icon {
  vertical-align: text-bottom; /* Aligns with text descender */
}
```

## Text Shadows (`text-shadow`)

Adds shadow effects to text:

```css
h1 {
  /* horizontal vertical blur color */
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
}

.neon {
  text-shadow: 0 0 5px #0ff, 0 0 10px #0ff;
}

.inset {
  color: white;
  text-shadow: 1px 1px 1px rgba(0,0,0,0.5);
}
```

## Text Decorations (`text-decoration`)

Controls underlines, overlines, and line-throughs:

```css
a {
  text-decoration: none; /* Remove underline */
}

.underline {
  text-decoration: underline wavy blue; /* Style */
}

.strike {
  text-decoration: line-through red double;
}

.custom-underline {
  text-decoration: underline;
  text-decoration-color: #f06;
  text-decoration-thickness: 3px;
  text-underline-offset: 4px; /* Distance from text */
}
```

## Text Transform (`text-transform`)

Controls text capitalization:

```css
.uppercase {
  text-transform: uppercase; /* ALL CAPS */
}

.lowercase {
  text-transform: lowercase; /* all small */
}

.capitalize {
  text-transform: capitalize; /* First Letters */
}

.small-caps {
  font-variant: small-caps; /* Small Caps */
}
```

## Text Indentation (`text-indent`)

Controls first line indentation:

```css
article p {
  text-indent: 2em; /* Traditional paragraph indents */
}

.hanging-indent {
  text-indent: -2em;
  padding-left: 2em;
}
```

## Text Overflow (`text-overflow`)

Handles clipped text:

```css
.truncate {
  white-space: nowrap;
  overflow: hidden;
  text-overflow: ellipsis; /* Shows ... */
}

.clip {
  text-overflow: clip; /* Simply cuts off */
}
```

## Practical Examples

### Newsletter Heading
```css
.newsletter-title {
  text-align: center;
  text-transform: uppercase;
  letter-spacing: 2px;
  text-shadow: 1px 1px 3px rgba(0,0,0,0.2);
  text-decoration: underline double #e63946;
}
```

### Button with Hover Effects
```css
.button {
  text-align: center;
  text-decoration: none;
  transition: all 0.3s;
}

.button:hover {
  text-shadow: 0 0 8px rgba(255,255,255,0.8);
  text-decoration: underline;
}
```

### Data Table Text
```css
.data-table td {
  text-align: right;
  vertical-align: middle;
}

.data-table th {
  text-align: left;
  text-decoration: underline dotted;
}
```

## Common Issues & Solutions

**Problem:** Uneven underline spacing
```css
.fix-underline {
  text-underline-offset: 0.2em;
}
```

**Problem:** Shadow making text blurry
```css
.clear-shadow {
  text-shadow: 1px 1px 0 rgba(0,0,0,0.5); /* Less blur */
}
```

**Problem:** Justified text gaps
```css
.even-justify {
  text-align: justify;
  word-spacing: -0.1em;
}
```

## Browser Support Notes

- `text-underline-offset` works in modern browsers
- Multiple `text-shadow` values have excellent support
- `text-decoration-thickness` needs prefixes in some browsers
```css
/* Fallback for older browsers */
.underline-compat {
  text-decoration: underline;
  -webkit-text-decoration-color: red;
  -webkit-text-decoration-style: wavy;
}
```
